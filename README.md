# NardiLens

`NardiLens` — это настольная утилита для Windows, которая отображает настраиваемый числовой оверлей (от 1 до 24) поверх всех окон. Приложение разработано, чтобы помочь игрокам в онлайн-нарды, позволяя им визуально пронумеровать пункты на игровой доске.

Приложение легко настраивается, поддерживает профили и управляется через иконку в системном трее.

## Основные возможности

  * **Числовой оверлей:** Отображение чисел от 1 до 24 в любых заданных пользователем точках экрана.
  * **Интерактивный режим настройки:** Удобный полноэкранный режим для расстановки номеров с помощью мыши.
  * **Полная кастомизация:**
      * Выбор семейства шрифтов.
      * Настройка размера шрифта (включая "горячую" настройку колесиком мыши в режиме конфигурации).
      * Выбор цвета текста.
      * Настройка цвета и толщины обводки для лучшей читаемости.
  * **Управление профилями:** Создавайте, переименовывайте, удаляйте и переключайтесь между несколькими профилями. Это позволяет сохранять разные раскладки (например, для разных игровых сайтов или разрешений экрана).
  * **Интеграция в трей:** Приложение работает в фоновом режиме, и им можно управлять через иконку в системном трее (показать/скрыть оверлей, открыть панель управления или выйти).
  * **Предпросмотр:** Окно настроек оформления показывает живой предпросмотр изменений.

## Как использовать

Этот репозиторий содержит исходный код. Для запуска в режиме разработки требуются зависимости; для конечного пользователя предназначен скомпилированный `.exe` файл (который, судя по структуре файлов, находится в папке `NardiLens - release`).

### Для пользователей (используя `NardiLens.exe`)

1.  Запустите `NardiLens.exe`.
2.  Иконка приложения появится в вашем системном трее (область уведомлений).
3.  Кликните правой кнопкой мыши по иконке, чтобы увидеть меню, или двойным кликом откройте "Панель управления".

### Для разработчиков (запуск из исходного кода)

1.  Убедитесь, что у вас установлен Python 3.
2.  Установите необходимую библиотеку:
    ```bash
    pip install PyQt6
    ```
3.  Запустите главный скрипт приложения:
    ```bash
    python overlay_app.py
    ```

## Первая настройка (Пошаговое руководство)

При первом запуске (или при создании нового профиля) оверлей не будет показан, так как координаты еще не заданы.

1.  Запустите приложение и откройте **"Панель управления"** (двойной клик по иконке в трее).
2.  Нажмите кнопку **"Настроить координаты"**.
3.  Экран потемнеет, и вы перейдете в режим настройки. В центре экрана появится инструкция.
4.  **Последовательно кликните левой кнопкой мыши** в тех 24 местах на экране, где вы хотите видеть номера (начиная с пункта 1 и заканчивая пунктом 24).
5.  **Во время настройки:**
      * **Левая кнопка мыши:** Поставить следующий номер.
      * **Правая кнопка мыши:** Отменить последнюю поставленную точку.
      * **Колесико мыши:** Увеличить или уменьшить размер шрифта "на лету".
      * **Клавиша ESC:** Выйти из режима настройки, не сохранив изменения.
6.  После 24-го клика режим настройки автоматически завершится, и координаты сохранятся в текущий профиль.
7.  Теперь вы можете использовать кнопку **"Показать оверлей" / "Скрыть оверлей"** (в главном окне или в меню трея).

## Настройка оформления

В главном окне "Панели управления" нажмите кнопку **"Настройки оформления..."**.
В этом окне вы можете:

  * Изменить шрифт.
  * Настроить размер шрифта и толщину обводки с помощью ползунков.
  * Выбрать цвет текста и цвет обводки.
  * Сбросить настройки оформления к значениям по умолчанию.

Все изменения сохраняются автоматически для **текущего активного профиля** при нажатии кнопки "Сохранить".

## Файл конфигурации

Все ваши профили, настройки и координаты автоматически сохраняются в файл `config.json`, который находится в той же директории, что и приложение.
